# TOEV
트위터 로그인 기반 투표 서비스
## 주의
본 프로젝트는 2018년 11월에 Node.js, HTML 및 CSS, OAuth, mySQL 연습용으로 제작되었습니다. 재기동이 가능하도록 코드를 일부 수정했습니다.
## 소개

SNS - 트위터 로그인으로 간단한 투표 사이트를 제작하였습니다.
